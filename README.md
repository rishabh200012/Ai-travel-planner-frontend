# 🌍 AI Travel Planner - Frontend

A modern AI-powered travel planner built with **Next.js** that helps users create personalized travel itineraries, estimate budgets, discover recommended hotels, and generate packing lists.

## 🚀 Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- Axios
- React Hot Toast

---

## ✨ Features

### 🔐 Authentication

- User Registration
- Email OTP Verification
- Login
- Logout
- Protected Dashboard

---

### ✈️ Trip Management

- Create New Trip
- View All Trips
- View Trip Details
- Delete Trip

---

### 🤖 AI Features

- Generate Complete Travel Itinerary
- Regenerate Individual Day
- Generate Packing List
- AI Estimated Budget
- AI Hotel Recommendations

---

### 📝 Itinerary Management

- Add Custom Activity
- Remove Activity
- Regenerate Single Day

### 💰 Budget Planner

- Transport
- Activities
- Total Budget

### 🏨 Hotel Suggestions

- Price Range
- Description

### 🎒 Packing List (CUSTOM FEATURE)

Generate destination-specific packing checklist using AI.

## 📦 Installation

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

## Environment Variables

Create a `.env.local`

